This project provides a full PDF generation engine for SironKYC results using Spring Boot and iText7. 
It parses KYC XML/JSON responses, processes watchlist matches, risk assessment, and personal data, 
and produces fully-formatted PDF reports with support for:

✓ Arabic and English fonts  
✓ Table-based structured layout  
✓ Header/Footer customization  
✓ Base64 encoded output  
✓ Attachments and history sections  
✓ Dynamic rendering of names, birthdays, IDs, nationalities, and addresses  

Suitable for AML, Compliance, KYC operations, and automated reporting workflows.
